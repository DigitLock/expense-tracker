# Expense Tracker

Expense Tracker is a personal and family budgeting project focused on tracking incomes, expenses, accounts, and basic reports with multi-currency support (RSD/EUR).

At this stage the repository contains **documentation for the MVP** (product and system requirements). Implementation will be added in later commits.

## Documentation

All documentation lives under the `Documentation/` folder:

- `Documentation/expense_tracker_brd.md` – **Business Requirements Document (BRD)**
- `Documentation/expense_tracker_srs_mvp.md` – **System Requirements Specification (SRS, MVP)**
- `Documentation/PDF/expense_tracker_brd.pdf` – BRD (PDF export)
- `Documentation/PDF/expense_tracker_srs_mvp.pdf` – SRS (PDF export)
